南开大学学堂云在线小助手(nkdx.xuetangx.com)
基于助教权限下学堂云教学辅助

===========================
主要功能：
	作业批量下载；
		1. 选择配置信息
		2. 自动将所有文件下载至工作目录（指定）/download下
	作业成绩批量上传；
		1. 选择配置信息
		2. 导出成绩模板
		3. 填写成绩，不要修改其他部分
		4. 上传成绩模板		
文件结构：
	程序根目录:
		code: 主程序
		packages: selenium安装包（mac） 
	工作目录:
		conf: 保存隐私配置（用户名 密码）
		download: 作业下载
		grade: 成绩模板
 
===========================
运行依赖： Python3.7 + Selenium
开发环境： Mac OS X 10.15
运行方法： 1. 安装所需依赖
	 2. 自动运行：运行 code/homeworks.py
	    调试运行：运行 code/homeworks.py /main2()
